DOM의 제어

html 구조를 html내에서 짜고 수정해도 되지만 js안에서 html 태그의 생성과 수정을 처리 할 수 있다. DOM API 를 활용하여 DOM 트리에 js 로 생성해주거나 수정하는 방식으로 html 의 컨트롤이 훨씬 유용해 진다.


node의 생성과 삽입

  • 태그 node 객체의 생성
