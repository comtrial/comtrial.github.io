블록요소

ex) div, h1, p

요소의 라인 전체(사용가능 최대 가로 너비)를 의미하고 제어한다. 가로 세로의 사이즈를 지정해 줄수있다.
그렇기 때문에 수직으로 쌓이게 되며 레이아웃을 작성할때 사용하는 요소


인라인 요소

ex) span, img

그저 요소 그 자체의 크기 만큼을 가져가는 요소이며, 가로와 세로의 사이즈를 설정 할 수가 없다. 그렇기에 수평으로 쌓기에 되며 TEXT를 작성 할 대 사용하는 요소이다.

cf) 위의 블록, 인라인 요소는 display 속성으로 서로 바꿔 줄 수 있다.

  • <a>

-anchor(외부의 주소에 닻을 내린다.)

-href 속성: 외부주소의 url 을 톨해 겅로를 알려줌

-rel : 현재 문서와 외부 링크와의 관계

-target: a tag 로 가는 페이지를 어디 tab에 띄울 것이냐(ex)_self, _blank)

-버튼 형식으로 많이 쓰이는데, 인라인 요소이기 때문에 display 값을 block 라인 으로 바꿔 줘야 한다.

-target=“_self” 가 기본 값으로 설정되어 있다.

-downlasd = “download” 이기 때문에 download 만 입력 해 주면 된다.

-같은 페이지 내에서의 이동도 가능 하다.
:id 값을 사용하여 <a href=“#id값”>
주소창에도 마지막 부분에 #title1 등의 id 값을 가진 페이지 내의 위치로 이동 하게 된다.

  • <span>

-div tag 와 용도가 같지만 display 요소가 인라인 요소이다.
