# CSS 단위 

* px 

해상도에 따른 차이가 있지만 주로 고정폭을 이야기 할 때 사용된다.

<br>
<!--more-->

* % 

보통 부모 요소의 크기로 상속되어 내려옴으로 부모 요소의 기준으로 작성 된다.

<br>

* em

자기 자신의 폰트 사이즈에 영향을 받는다.  그리고 그 자신의 폰트 사이즈는 부모요소 한테 상속받는다.

```css
.container {
	width: 60em;
	font-size: 10px;
}


=>

.container {
	width: 600px;
	font-size: 10px;
}
```


자기 자신의 폰트 크기가 지정 되어 있지 않다면 조상요소 폰트 사이즈에 영향을 받기 때문에  유지 보수가 쉽지 않다.

* rem

루트를 타고 올라가면 결국 html 태그이기 때문에 rem이란 html 이 작성된  폰트 사이즈 기준으로 em 이 잡히게 된다. 

html 의 폰트 사이즈를 지정해 준 후 body 태그를 건들여주면 


* vw. wh

ViewPort가 기준이되는 단위이다. v의 기준으로 width 와 height를 정할 수 있게 된다. 1~100까지의 값을 가지게 된다.

* vmim, vmax
 
view의 가로세로중 더 크거나 작은 값을 기준으로 하는 단위이다.
