### Git에 commit 하기

우선 원하는 파일을 선택하여 전에 만들었던 빈 로컬 Git 저장소에 저장을 해야 한다. 
<br>
원하는 하나의 파일만의 선택은 
```bash
$git add readme.md
```
<!--more-->
이고
<br>

폴더의 모든 파일의 선택 은 
```bash
$git add .
```
이다.
<br>

이렇게 추가한 파일들을 commit, 즉 하나의 버전으로 만들어주는 과정이 필요하다. 
```bash
$git commit -m "프로젝트 설명"
```

의 과정을 통해 앞서 선택한 파일들을 하나의 버전으로 만들어 commit 할 수 있게 된다.
<br>

이렇게 생성된 커밋을 확인 하는 방법은
```bash
$git log
```

<br>

로그에 찍혀서 알 수 있듯  상태의 시점과 파일의 수정 내용이 보이게 되는데 , 사실 git 의 장점은 개인 프로젝트를 할 때 보다, 여러 명에서 프로젝트를 진행할때 코드의 변경 사항과 
업데이트 내용 등의 확인에서 더욱 두드러지게 된다.
